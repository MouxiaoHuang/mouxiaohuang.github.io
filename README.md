# mouxiaohuang.github.io

Personal homepage for [Mouxiao Huang](https://mouxiaohuang.github.io), built with [Jekyll](https://jekyllrb.com/) and hosted on GitHub Pages.

## Local preview

```bash
gem install jekyll bundler
jekyll serve --livereload
```

Open http://127.0.0.1:4000/

## Structure

- `index.html` — homepage
- `blogs/` — blog section (currently empty)
- `tools/` — utilities (e.g. salary calculator)
- `images/` — site assets

## Blogs

See `blogs/posts/README.md` for how to add new posts. Templates are in `blogs/templates/`.
